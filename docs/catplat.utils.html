catplat.utils.slab module

This module consists of the utilities functions used in SlabBuilder.

catplat.utils.slab.compact_bulk_in_unitcell(atoms) → GratomsWrapper[source]

Translates atoms in the bulk so that the bulk structure is ‘nice’ and compact. This prevents creation of multiple slabs if the bulk structure has some roughness.

catplat.utils.slab.get_avg_coord_num(slab) → float[source]

Returns the mean of all surface atoms’s avg_coord_num. i.e., fcc111 returns 9.0 and fcc100 returns 8.0

catplat.utils.slab.get_unique_coordinates_cartesian_tol(atoms, axis=2, tag=False, tol=0.4, return_tag=False)[source]

Modified function from catkit.gen.utils.get_unique_coordinates.

Changelog: 1. tol changed to cartesian coords, i.e. units of Angstroms, instead of fraction coordinates. 2. Use z component of 3rd cell vector only for determining cell length 3. Added return_tag flag to return the tags instead of values
